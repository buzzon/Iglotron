⚡ БЫСТРАЯ УСТАНОВКА:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Установите vcpkg (если еще не установлен):
   
   cd C:\
   git clone https://github.com/Microsoft/vcpkg.git
   cd vcpkg
   .\bootstrap-vcpkg.bat
   setx VCPKG_ROOT "C:\vcpkg"

2. Установите зависимости:
   
   vcpkg install glfw3:x64-windows
   vcpkg install imgui[glfw-binding,opengl3-binding]:x64-windows
   vcpkg install glad:x64-windows

3. Соберите проект:
   
   cd camera_app
   build.bat

4. Запустите:
   
   cd build\Release
   camera_app.exe

❓ ЧАСТЫЕ ПРОБЛЕМЫ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Проблема: "CMake Error: Could not find OpenCV"
Решение:  Проверьте путь C:\opencv\build или измените в CMakeLists.txt

Проблема: "Failed to open camera"
Решение:  Закройте Skype/Teams/Zoom, проверьте Privacy → Camera в Windows

Проблема: "GPU not available"
Решение:  Приложение автоматически использует CPU. Все работает!

Проблема: Отсутствуют DLL при запуске
Решение:  Скопируйте OpenCV DLL из C:\opencv\build\x64\vc16\bin
