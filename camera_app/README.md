# Приложение с камерой

Qt приложение, которое захватывает видео с камеры и отображает его на экране с двумя кнопками.

## Требования

- Qt 6 (с модулями Multimedia и MultimediaWidgets)
- Компилятор с поддержкой C++17
- CMake 3.16+ или qmake

## Сборка с помощью CMake

```bash
cd camera_app
mkdir build
cd build
cmake ..
make
./camera_app
```

## Сборка с помощью qmake

```bash
cd camera_app
qmake
make
./camera_app
```

## Возможности

- Захват видео в реальном времени с камеры
- Виджет отображения видео
- 2 кнопки (Кнопка 1, Кнопка 2) - в данный момент ничего не делают, как и требовалось

## Примечания

- Убедитесь, что в вашей системе есть рабочая камера
- В Linux может потребоваться предоставить разрешения на доступ к камере
- Приложение будет использовать камеру по умолчанию
